## 🛠️ 安装指南

本插件为非官方扩展，需以“开发者模式”安装。

1.  **下载仓库**：将本 GitHub 仓库克隆或下载到本地：
    ```bash
    git clone [https://github.com/你的用户名/你的仓库名.git](https://github.com/你的用户名/你的仓库名.git)
    ```
    或直接点击 `Code` -> `Download ZIP` 下载。

2.  **解压文件**：如果下载的是 ZIP 文件，请将其解压到一个你容易找到的文件夹。

3.  **打开 Chrome 扩展程序页面**：
    * 在 Chrome 浏览器中，访问 `chrome://extensions`。
    * 或点击浏览器右上角的三个点 -> `更多工具` -> `扩展程序`。

4.  **启用开发者模式**：
    * 在扩展程序页面的右上角，打开 `开发者模式` 开关。

5.  **加载已解压的扩展程序**：
    * 点击左上角的 `加载已解压的扩展程序` 按钮。
    * 选择你之前解压的插件文件夹 (`NJU-Login-Pro-main` 或你克隆的仓库文件夹)。

6.  **固定插件**：
    * 安装成功后，插件会出现在浏览器工具栏的拼图图标中。点击拼图图标，然后点击 `NJU Login Pro` 旁边的**图钉**图标，将其固定在工具栏上，方便快速访问。

## 💡 使用方法

### 自动登录

1.  **配置账号信息**：
    * 点击浏览器工具栏上的插件图标，切换到「**🔐 登录助手**」Tab。
    * 输入学号/工号和密码，按需开启各开关，点击"保存"。

2.  **自动登录**：
    * 访问对应的南大系统登录页面，插件自动识别验证码并填充登录。
    * **统一身份认证**：自动识别图形验证码 → 自动填充 → 自动点击登录
    * **选课系统**：自动填充账号密码 → 自动识别点击式验证码 → 自动登录

### 选课助手（自动抢课）

1.  在浏览器中打开选课系统并登录，进入**体育课**或目标选课页面。
2.  点击插件图标，切换到「**📚 选课助手**」Tab。
3.  在文本框中输入目标课程名称（每行一个，支持模糊匹配），例如：
    ```
    仙林游泳初级
    仙林足球初级
    ```
4.  选择刷新间隔（建议 5 秒以上，避免过于频繁）。
5.  点击「**▶ 开始监控**」：
    * 如果课程有余量，插件自动点击「选择」按钮并确认弹窗，立即完成选课。
    * 如果课程已满，持续监控，一旦有空位立即尝试抢课。
    * 所有目标课程抢到后自动停止，日志区实时显示每轮检测结果。
6.  点击「**⏹ 停止监控**」可随时手动停止。

## 🌐 支持的网站

### 1. **统一身份认证系统** 
   - 网址：`https://authserver.nju.edu.cn/`
   - 验证码：图形验证码（4位字母数字）
   - 功能：✅ 自动识别、自动填充、自动登录
   - 准确率：95%+（Tesseract.js OCR）

### 2. **选课系统** (v3.0 新增)
   - 网址：`https://xk.nju.edu.cn/`
   - 验证码：**点击式验证码**（依次点击指定中文字符）
   - 功能：✅ 自动填充账号密码 + ✅ 自动识别验证码 + ✅ 自动点击登录
   - 技术：中文 OCR 识别 + 字符位置定位 + 模拟点击

### 3. **选课助手 · 自动抢课** (v3.1 新增)
   - 入口：插件 Popup → 「📚 选课助手」Tab
   - 功能：✅ 模糊匹配课程名 + ✅ 自动点击"选择"按钮 + ✅ 自动确认弹窗 + ✅ 满课持续监控
   - 选择按钮：`.cv-choice`；确认按钮：`.cv-sure.cvBtnFlag`
   - 刷新间隔：可选 1 / 2 / 3 / 5 / 10 / 15 / 30 秒（最小 1 秒）
   - 支持同时监控多门课程，全部抢到后自动停止

## ⚙️ 技术栈

* **HTML/CSS/JavaScript**：构建插件界面和核心逻辑。
* **Chrome Extension API**：`chrome.storage` 本地数据存储；`chrome.tabs` + `chrome.runtime.sendMessage` 实现 Popup 与页面脚本的实时通信。
* **Tesseract.js**：浏览器端的 OCR 识别库，用于处理验证码图片。

## ⚠️ 注意事项

* 本插件仅为学习交流和个人便利而开发，请勿用于非法用途。
* 数据仅在您的浏览器本地存储，不会上传到任何服务器。但请注意，在公共电脑上使用自动填充功能仍存在安全风险。
* 若南京大学统一身份认证页面结构发生变化，插件可能需要更新以适配新的元素选择器。
* Tesseract.js 识别率受验证码复杂度和清晰度影响，极端情况下可能需要手动输入。
